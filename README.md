# Gensokyo
>充满幻想的WordPress主题
## 依赖
```
  $ npm install
  $ npm install -g gulp
  $ gulp release
```
## 提醒
本主题不适合新手使用，使用前请确保具有前端 PHP node.js的基础。 
顶部头像：后台->外观->顶部->上传  
依赖的插件（必装）：Post Ratings、WP-PostViews  
js采用webpack压缩 css采用glup压缩使用前请先浏览目录
